## 🧩 What is PaperArtifacts?

PaperArtifacts brings the beloved [Artifacts mod](https://modrinth.com/mod/artifacts) experience to **Paper & Spigot** servers as an addon for [CuriosPaper](https://modrinth.com/plugin/curiospaper).

It adds **45+ unique magical items** with custom textures that players can discover through **dungeon loot, mob drops, and abandoned campsites** — underground structures that generate naturally in your world. Some chests in those campsites might be **Mimics** — hostile mobs disguised as chests that attack when opened!

<p align="center">
  <img src="https://cdn.modrinth.com/data/w0FXMugd/images/3fc404ab62e3e6ce238fddb17bb8127e5342c903_350.webp" alt="All available artifact items stored in a chest" width="500">

Every item is fully configurable, has a custom resource pack model, and works out of the box.

---

## ⚡ Key Features

- ⚔️ **45+ artifacts** across head, necklace, belt, hands, and feet slots
- 🎨 **Custom resource pack** with unique textures for every item — auto-merged with CuriosPaper
- 🏕️ **Abandoned campsites** that generate underground with loot, furniture, and traps
- 👹 **Mimics** — chest-shaped mobs hiding in campsites that attack when you try to open them
- 🗺️ **Loot table integration** — artifacts appear in vanilla dungeon chests, temples, bastions, and more
- 🧟 **Mob drops** — themed artifacts drop from specific mobs
- ⚙️ **Fully configurable** — enable/disable and tune every item and feature in `config.yml`
- 🔧 **Hot reload** — `/artifacts reload` applies changes instantly

## 🏕️ Campsites & Mimics

**Abandoned campsites** generate underground in caves. They contain beds, crafting stations, furnaces, barrels, and loot chests — some with artifacts inside.

<p align="center">
  <img src="https://cdn.modrinth.com/data/w0FXMugd/images/9c730693f9f96aa605a03521213d4bfa74860b80_350.webp" alt="A naturally spawned abandoned campsite underground" width="500">
</p>

But be careful — some chests are actually **Mimics**. They look like normal chests, but when you open one, it transforms into a hostile mob with custom animations and sounds. Mimics only exist inside campsites and won't activate unless you interact with them. Defeat one to earn a random artifact!

<p align="center">
  <img src="https://cdn.modrinth.com/data/w0FXMugd/images/d21c8f4b61bfe7c6699e021d8b1bd4afc0de5e79_350.webp" alt="A mimic attacking the player" width="400">
</p>

Chests can also be **trapped** with TNT underneath. Exploration is rewarding — but risky.

## 📦 How to Find Artifacts

| Method | Description |
|---|---|
| 🗺️ **Loot Chests** | Found in dungeons, temples, bastions, end cities, mineshafts, and more |
| 🧟 **Mob Drops** | Specific mobs drop themed artifacts (e.g. Drowned → Flippers) |
| 🏕️ **Campsites** | Underground structures with loot chests and a chance for artifacts |
| 👹 **Mimics** | Kill a mimic in a campsite for a guaranteed artifact |
| 📋 **Commands** | `/artifacts give <item> [player]` for admins |

## 📥 Getting Started

1. Install [CuriosPaper](https://modrinth.com/plugin/curiospaper) on your server
2. Drop `Artifacts.jar` into your `plugins/` folder
3. Start the server — done!

Artifacts will start appearing in dungeon chests, dropping from mobs, and generating in campsites across your world. Use `/baubles` to open the accessory menu and equip them.
